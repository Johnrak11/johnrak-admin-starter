What It Can Do

The Johnrak AI is a context-aware assistant designed to represent you. It has real-time access to your database, meaning it "knows" everything you've added to your portfolio.

Core Knowledge Base:

  • Profile & Identity: Knows who "Vorak" is, your summary, headline, and contact links.
  • Projects: Can list your projects, explain their tech stacks, and describe their features.
  • Skills: Knows what you are good at (e.g., "Are you good at Laravel?").
  • Experience: Can recount your work history, previous companies, and roles.

How It Works (Architecture)

Unlike standard chatbots, this system uses a technique called Full Context Injection.

  1. User asks a question: "What projects did you build with Vue?"
  2. Data Retrieval: The system instantly fetches your entire portfolio (Profile, Projects, Skills, Experience) from the MariaDB database.
  3. Prompt Assembly: It constructs a "System Instruction" that includes your persona ("You are Johnrak...") and the raw JSON data of your portfolio.
  4. AI Processing: It sends this package to Google Gemini (Flash Model).
  5. Response: Gemini analyzes your specific data and answers the user's question accurately, without hallucinating.

Setup & Configuration

The system is designed to be low-maintenance and free to run (using Gemini's free tier).

1. Get an API Key

Go to Google AI Studio and create a free API key.

2. Configure in Admin

  1. Navigate to Settings → AI Integration.
  2. Paste your key into the Google Gemini API Key field.
  3. Click Save. The system automatically updates the .env file and clears the cache.

Example Conversations

Try asking the assistant these questions:

  • "Who is Vorak?" (Fetches from Profile)
  • "List my top 3 skills." (Fetches from Skills)
  • "Tell me about the admin dashboard project." (Fetches from Projects)
  • "Where did Vorak work in 2024?" (Fetches from Experience)

Troubleshooting

"Vorak's AI is resting..."

This generic error message means the backend failed to get a response from Google. Common causes:

  • Missing API Key: Check Settings.
  • Quota Exceeded: You may have hit the free tier rate limit (15 requests/minute). Wait a moment and try again.
  • Network Issue: The server couldn't reach Google's API.

"I haven't added that experience..."

The AI is strictly instructed not to lie. If you ask about "React Native" but haven't added it to your Skills list in the database, the AI will honestly say it doesn't know about it. To fix this, simply add the skill or project in your portfolio admin.
